# Enthiran Jump — AI Powered Custom Doodle Jump Game

A custom Doodle Jump-inspired browser game themed around the Tamil sci-fi movie Enthiran (Robot).

This project combines AI-assisted development, game mechanics, cinematic visuals, sound effects, and frontend web technologies to create a unique movie-inspired gameplay experience.

---

## Project Inspiration

Instead of building a generic game, I wanted to create something personal and creative by combining:

- Doodle Jump mechanics
- Enthiran movie theme
- Rajinikanth-inspired robot character
- Cinematic visuals
- Sound effects
- AI-assisted coding workflow

---

## Features

- Custom movie-inspired menu screen
- Start game functionality
- High score tracking system
- Left and right character movement
- Infinite jumping gameplay
- Game over system
- Background music and sound effects
- Futuristic visual atmosphere

## Challenges Faced

During development, I encountered several real-world frontend development issues:

### Image Loading Errors
Incorrect image URLs caused rendering failures inside the canvas system.

### Canvas Rendering Issues
Broken assets triggered:
`CanvasRenderingContext2D.drawImage` errors.

### Background Rendering Problems
Poster and gameplay backgrounds initially failed due to formatting mistakes.

### Browser Audio Restrictions
Autoplay audio was blocked by browsers and required user-triggered interaction handling.
